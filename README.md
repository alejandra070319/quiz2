# Unilibre Front-End

Plataforma profesional de Unilibre desarrollada con React. Una interfaz moderna y funcional con barra lateral navegable.

## Características ✨

- ✅ Barra lateral (Sidebar) elegante y responsiva
- ✅ Navegación funcional entre secciones
- ✅ Diseño moderno con gradientes y animaciones
- ✅ Interfaz totalmente responsiva
- ✅ Componentes reutilizables en React Router
- ✅ Formulario de contacto funcional
- ✅ Diseño profesional y atractivo

## Tecnologías Usadas 🚀

- **React 18.2.0** - Librería JavaScript para construcción de interfaces
- **React Router DOM 6.4.0** - Navegación entre páginas
- **Lucide React** - Iconos modernos y optimizados
- **CSS3** - Estilos y animaciones personalizadas
- **Poppins Font** - Tipografía moderna

## Instalación 📦

Clona el repositorio:

```bash
git clone https://github.com/alejandra070319/quiz2.git
cd quiz2
```

Instala las dependencias:

```bash
npm install
```

## Uso 💻

Para iniciar el servidor de desarrollo:

```bash
npm start
```

El proyecto se abrirá en `http://localhost:3000`

### Construir para producción:

```bash
npm run build
```

## Páginas Disponibles 📄

1. **Home** - Página principal con hero section y características
2. **Catálogo** - Categorías de productos con búsqueda
3. **Productos** - Listado de productos con filtros
4. **Contacto** - Formulario de contacto funcional
5. **Quiénes somos** - Información sobre la empresa, equipo y estadísticas
